# 🤖 World-Class AI Assistant (GPT-4o + Vision + DALL-E 3)

A premium, multimodal Discord AI assistant upgraded from GPT-3 to the latest **GPT-4o** and **DALL-E 3** models. This bot can chat, "see" images, and generate high-quality art directly in your Discord server.

---

## 💎 Elite Features

### 🧠 Advanced Intelligence
- **GPT-4o Engine:** Powered by the world's smartest AI model for natural, witty, and accurate conversations.
- **Vision Capabilities:** Attach an image to your message and ask the AI to analyze, describe, or explain it.
- **Context Awareness:** Remembers the last 10 messages in the channel for seamless, flowing conversations.

### 🎨 Creative Power
- **DALL-E 3 Integration:** Use the `!image <prompt>` command to generate stunning, high-resolution artwork instantly.
- **Premium UI:** Every response is wrapped in a sleek, gold-themed embed with decorative dividers.

### ⚡ Performance
- **Optimized for Railway:** Lightweight and designed to run 24/7 with minimal resources.
- **Smart Filtering:** Automatically ignores other bots and focuses on your dedicated AI channel.

## 🚀 Setup & Deployment

### 1. Environment Variables
Add these to your Railway or local `.env` file:

| Variable | Description |
|---|---|
| `BOT_TOKEN` | Your Discord Bot Token |
| `OPENAI_KEY` | Your OpenAI API Key (Must have GPT-4o access) |
| `BOT_CHANNEL` | The ID of the channel where the AI will respond |

### 2. Installation
```bash
npm install
```

### 3. Start the Bot
```bash
npm start
```

---

## 🛠️ Commands

- **Just Chat:** Type anything in the designated AI channel to talk to GPT-4o.
- **Vision:** Attach an image and ask a question to use the Vision engine.
- **`!image <prompt>`:** Generate high-quality AI art using DALL-E 3.
